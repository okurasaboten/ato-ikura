# iPhoneで使う手順

このアプリは、iPhoneのSafariで開いて「ホーム画面に追加」すると毎日使いやすくなります。
計算期間は毎月12日〜翌月11日です。固定費に登録した金額は、どの期間でも自動で差し引かれます。

## 1. 公開する

おすすめは GitHub Pages です。

1. GitHubで新しいリポジトリを作る
2. `work` フォルダ内のファイルをアップロードする
3. リポジトリの `Settings` → `Pages` を開く
4. `Deploy from a branch` を選ぶ
5. `main` ブランチ、`/root` を選んで保存する
6. 表示されたURLをiPhoneのSafariで開く

## 2. ホーム画面に追加する

1. iPhoneのSafariでアプリURLを開く
2. 共有ボタンを押す
3. 「ホーム画面に追加」を選ぶ
4. 名前が「あといくら」になっていることを確認して追加する

## 3. 使うときの注意

- データはiPhoneのSafari内に保存されます。
- Safariの履歴やWebサイトデータを消すと、アプリの記録も消えることがあります。
- 機種変更やバックアップ用に、将来的にはCSV書き出し機能を追加すると安心です。
